Installer Node.js

Vi kommer til å bruke React i workshopen. Det er et rammeverk som er basert på Node.js

  1. Last ned Node.js her

Andre måter å installere på

  • Hvis du bruker Homebrew kan du installere med

    $ brew install nodejs

  • På Linux kan du installere med apt eller dnf eller en annen "package manager" avhengig av hva slags Linux-distribusjon du bruker

    $ sudo dnf install nodejs

Sjekk at den er installert

Sjekk at Node.js er installert ved å kjøre følgende kommando i Terminalen:

$ npm
