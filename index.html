1 Installer Node.js

Vi kommer til å bruke React i workshopen. Det er et rammeverk som er basert på Node.js

  1. Last ned Node.js her
  2. Dobbeltklikk på msi-filen hvis du er på Windows eller pkg-filen hvis du er på macOS og følg instruksjonene for å installere

1.1 Andre måter å installere på

N.B.: Dette er bare for de spesielt interessert!

  • Hvis du bruker Homebrew kan du installere med

    $ brew install nodejs

  • På Linux kan du installere med apt eller dnf eller en annen "package manager" avhengig av hva slags Linux-distribusjon du bruker

    $ sudo dnf install nodejs

1.2 Sjekk at den er installert

Sjekk at Node.js er installert ved å kjøre følgende kommando i Terminalen:

$ npm

2 Sjekk at create-react-app fungerer

Vi kommer til å bruke Create React App for å pang-starte prosjektet vårt!

  1. I de neste stegene kommer du til å trenge en terminal. Hvis du har macOS, åpne Spotlight ved å trykke Command+Space, skriv "Terminal" og trykk enter.

    Hvis du er på Windows har du kanskje en terminal i editoren din (f.eks. Visual Studio Code, PyCharm).

  2. Bruk npx til å sjekke at du får startet et React-prosjekt

    $ npx create-react-app min-testapp
